WordPress core security class: Protect your wordpress user inputs from the usual array of attack vectors

== Description ==

= Pareto Security Features =

* No customisation needed
* Protects against database injections
* Protects against _POST injection of malicious code
* Processes the REQUEST_URI, QUERY_STRING, _GET, _POST, _COOKIE and browser user-agents to detect values with a blacklisted format.

== Installation ==

* <strong>Automated Setup Steps</strong>

1. Upload `/pareto-security/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
